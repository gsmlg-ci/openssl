# openssl

[![Build](https://github.com/gsmlg-ci/openssl/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/openssl/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/openssl`
- `ghcr.io/gsmlg-dev/openssl`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/openssl:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/openssl:latest
```

## Build

```bash
docker build -t gsmlg/openssl .
```
